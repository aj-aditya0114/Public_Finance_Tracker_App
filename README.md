# Public_Finance_Tracker_App
A simple **Spring Boot** web application to help users track their income, expenses, and budgets. This project uses **Thymeleaf** for the frontend and **Spring MVC** with **JPA/Hibernate** for backend.

---

## Features

- **User Registration & Login**
  - Secure login and registration system
  - Stores user credentials

- **Income Management**
  - Add income sources (salary, bonus, freelance)
  - Track recurring and one-time incomes

- **Expense Tracking**
  - Record daily expenses with categories
  - Add descriptions for better organization

- **Budget Setting**
  - Set monthly budgets for each category
  - View progress and overspending alerts

- **Transaction History**
  - View past transactions
  - Search and filter by date, type, or category

- **Dashboard**
  - Overview of total income, total expenses, and savings
  - Personalized greeting with user’s name

---

## Technologies Used

- Java 17
- Spring Boot 3
- Spring MVC

- Thymeleaf
